## ⚠️ 重要提示

当前开发环境网络无法访问GitHub（连接超时），需要您在**其他电脑**或使用**其他网络**执行推送。

## 推送步骤

### 方法一：在其他电脑/网络下执行

1. 将项目文件夹复制到能访问GitHub的电脑
2. 或使用U盘/网盘传输项目文件
3. 在该电脑上打开命令行，执行：

```powershell
cd 项目文件夹路径
git push -u https://github.com/VKingGeeker/ai-marketing-platform.git master
```

### 方法二：使用VS Code（推荐）

1. 用VS Code打开项目文件夹
2. 点击左侧Source Control图标
3. 点击"Push"按钮推送

### 方法三：手动压缩传输

1. 压缩项目文件夹
2. 复制到能上网的电脑
3. 解压后执行推送命令

# AI 营销内容生成平台 - 项目规范

## 1. 项目概述

- **项目名称**: AI 营销内容生成平台 (AI Marketing Content Generator)
- **项目类型**: Web 单页应用 (SPA)
- **核心功能**: 为中小餐饮店、零售店提供 AI 营销文案、产品描述、社交媒体内容的智能生成服务
- **目标用户**: 中小餐饮店、零售店店主

## 2. UI/UX 规范

### 2.1 布局结构

- **顶部导航栏**: 固定高度 60px，包含 Logo、导航菜单、用户头像
- **主体内容区**: 左侧边栏 (可选) + 主内容区域
- **页面结构**:
  - 首页: 3 个功能入口卡片网格布局
  - 生成页面: 左右分栏布局 (左侧表单 40%，右侧结果 60%)
  - 历史记录: 表格/列表布局
- **响应式断点**:
  - 移动端: < 768px (单列布局)
  - 平板: 768px - 1024px (双列)
  - 桌面: > 1024px (完整布局)

### 2.2 视觉设计

- **主色调**:
  - 主色: #4F46E5 (靛蓝色)
  - 辅色: #10B981 (翡翠绿)
  - 强调色: #F59E0B (琥珀色)
  - 背景色: #F8FAFC (浅灰白)
  - 卡片背景: #FFFFFF
  - 文字主色: #1E293B
  - 文字次色: #64748B

- **字体**:
  - 标题: "Noto Sans SC", sans-serif, 700 weight
  - 正文: "Noto Sans SC", sans-serif, 400 weight
  - 字号: H1: 32px, H2: 24px, H3: 18px, Body: 14px, Small: 12px

- **间距系统**:
  - 基础单位: 4px
  - 组件间距: 16px / 24px / 32px
  - 页面边距: 24px (桌面) / 16px (移动)

- **视觉效果**:
  - 卡片阴影: 0 4px 6px -1px rgba(0, 0, 0, 0.1)
  - 悬停阴影: 0 10px 15px -3px rgba(0, 0, 0, 0.1)
  - 圆角: 8px (卡片) / 4px (按钮) / 12px (大卡片)
  - 过渡动画: 0.3s ease-in-out

### 2.3 组件规范

- **按钮**:
  - 主按钮: 背景 #4F46E5, 文字白色, 圆角 6px
  - 次按钮: 边框 #4F46E5, 文字 #4F46E5
  - 悬停: 亮度增加 10%
  - 禁用: 透明度 50%

- **输入框**:
  - 边框: 1px solid #E2E8F0
  - 聚焦: 边框 #4F46E5, 阴影 0 0 0 3px rgba(79, 70, 229, 0.1)
  - 圆角: 6px
  - 高度: 40px

- **卡片**:
  - 背景: 白色
  - 边框: 1px solid #E2E8F0
  - 圆角: 12px
  - 内边距: 24px

- **下拉选择**:
  - 使用 Element Plus Select 组件
  - 风格与输入框一致

## 3. 功能规范

### 3.1 首页

- **功能入口卡片** (3个):
  1. 营销文案生成 - 图标: document, 描述: "快速生成吸引人的营销文案"
  2. 产品描述生成 - 图标: goods, 描述: "为商品打造吸引人的描述"
  3. 社交媒体 - 图标: connection, 描述: "生成社交媒体推广内容"
- **卡片交互**: 悬停上浮效果，点击跳转对应页面

### 3.2 营销文案生成页面

- **左侧表单**:
  - 店铺名称 (输入框)
  - 经营类型 (下拉选择: 餐饮/零售/服务/其他)
  - 营销主题 (下拉选择: 开业促销/会员日/节日活动/新品推广/清仓甩卖)
  - 文案风格 (下拉选择: 亲切温暖/专业正式/活泼俏皮/高端大气/简洁直白)
  - 提交按钮

- **右侧结果区**:
  - AI 模拟生成 3-5 条文案卡片
  - 每条文案包含: 文案内容、操作按钮组
  - 操作: 复制、编辑、收藏
  - 空状态提示

### 3.3 产品描述生成页面

- **左侧表单**:
  - 商品名称 (输入框)
  - 商品类别 (下拉选择)
  - 商品特点 (多行文本框)
  - 目标客群 (下拉选择)
  - 提交按钮

- **右侧结果区**: 同营销文案页面结构

### 3.4 社交媒体生成页面

- **左侧表单**:
  - 平台类型 (多选: 微信/微博/抖音/小红书)
  - 内容主题 (输入框)
  - 目标受众 (下拉选择)
  - 提交按钮

- **右侧结果区**: 同上结构，按平台分组显示

### 3.5 历史记录页面

- **列表展示**:
  - 时间戳
  - 生成类型
  - 摘要内容
  - 操作 (查看/删除)
- **分页**: 每页 10 条
- **筛选**: 按类型筛选

## 4. 技术规范

### 4.1 技术栈

- Vue 3 + Composition API
- Vite 构建工具
- Element Plus UI 组件库
- Vue Router 4
- localStorage 存储
